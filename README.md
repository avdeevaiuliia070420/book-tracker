# Трекер прочитанных книг
Консольное приложение для ведения списка прочитанных книг.

## Возможности
- Добавление книги с автором, названием, оценкой (1–5) и датой
- Просмотр всех книг
- Расчёт средней оценки
- Статистика по авторам (количество книг)
- Удаление книг по номеру

## Запуск
```bash
python main.py
